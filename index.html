🧠 Artificial Neural Network

Click each card to learn the key term.

Input Layer
Receives the input features from the outside world and passes them to the network.
Hidden Layer
Processes information and learns patterns from the input data.
Output Layer
Produces the final prediction or classification.
Neuron
The basic processing unit of an artificial neural network.
Weight
Represents the importance of an input connection.
Bias
A constant value added to improve learning flexibility.
Activation Function
Determines whether a neuron should activate. Examples: ReLU, Sigmoid, Tanh.
Forward Propagation
Data moves from the input layer through hidden layers to the output.
Backpropagation
Updates weights by minimizing prediction errors.
Epoch
One complete pass of the training dataset through the neural network.
Learning Rate
Controls how much weights change during training.
Prediction
The final output generated by the trained neural network.
